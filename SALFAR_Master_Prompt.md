# SALFAR Sickle Aid Initiative - Complete Website Master Prompt

## Project Overview
Create a comprehensive, production-ready website for SALFAR (Salfar Sickle Aid Initiative), a Nigerian non-profit organization dedicated to supporting Sickle Cell Disease (SCD) warriors. The website should be modern, responsive, and professionally designed with Google's color scheme and excellent user experience.

## Technical Stack
- **Framework**: React 18+ with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Deployment**: Netlify-ready

## Design Requirements
- **Color Scheme**: Google colors (Red: #EA4335, Orange: #FB8C00, Green: #34A853, Blue: #4285F4, Yellow: #FBBC04)
- **Typography**: Inter font family, clean and readable
- **Layout**: Responsive design with mobile-first approach
- **Animations**: Smooth transitions, fade-in effects, hover states
- **Images**: Use Pexels stock photos for content imagery
